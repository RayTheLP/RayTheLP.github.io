# 範例文章

這是一個範例文章，你可以刪除它並添加你自己的文章。

## 如何添加新文章

1. 在 `posts.json` 中添加新的文章條目
2. 在 `posts/` 資料夾中創建對應的 Markdown 文件
3. 更新文章內容

## Markdown 語法支援

- **粗體文字**
- *斜體文字*
- `程式碼`
- [連結](https://example.com)

```python
# 程式碼區塊
print("Hello, World!")
```

## 文章結構

每篇文章在 `posts.json` 中需要包含以下欄位：

- `id`: 文章唯一識別碼
- `title`: 文章標題
- `date`: 發布日期 (YYYY-MM-DD 格式)
- `category`: 文章分類
- `tags`: 文章標籤 (陣列)
- `excerpt`: 文章摘要
- `content`: 文章內容 (Markdown 格式)
